class

record-dc% : class?

  superclass: object%

  extends: dc<%>

record-dc%对象记录绘图操作以重放到其他绘图上下文中。记录的绘图操作可以通过get-recorded-procedure过程提取为一个过程，也可以通过get-recorded-datum数据提取为一个数据（可以用write打印并用read重新创建）。

绘制录制的操作时，目标绘图上下文的笔、画笔、字体、文本、背景、文本背景和文本前景不会影响录制的操作。目标图形上下文的转换、alpha和剪切区域使用录制操作中的设置进行组合（例如，设置剪切区域的录制操作实际上会在重放录制的命令时与图形上下文的剪切区域相交）。重播重新编码的命令后，目标图形上下文中的所有设置（如剪切区域或当前字体）都与重播前相同。

constructor

(new record-dc%    
    [[width width]    
    [height height]]) → (is-a?/c record-dc%)
  width : (>=/c 0) = 640
  height : (>=/c 0) = 480

创建新的录制DC。可选的width和height参数决定了在记录get-size的结果；width和height参数不剪辑绘图。

method

(send a-record-dc get-recorded-datum) → any/c

将记录的绘图提取到一个值，该值可以用write打印，也可以用read重新读取。使用recorded-datum->procedure将基准转换为绘图程序。

method

(send a-record-dc get-recorded-procedure)

 → ((is-a?/c dc<%>) . -> . void?)

将记录的绘图提取到可应用于其他DC的过程中，以便将绘图命令重放到给定DC。

get-recorded-procedure方法比get-recorded-datum和recorded-datum->procedure的合成方法更有效。
