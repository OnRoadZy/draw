28 签名和单元

racket/draw/draw-sig和racket/draw/draw-unit库定义draw^签名和draw@实现。

28.1 绘图单元

 (require racket/draw/draw-unit) package: draw-lib

value

draw@ : unit?

重新导出racket/draw的所有导出。

28.2 绘图签名

 (require racket/draw/draw-sig) package: draw-lib

signature

draw^ : signature

包括通过racket/draw导出的所有标识符。
